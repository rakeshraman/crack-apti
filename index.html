Basic Technical Questions

Tell me something about OOPS concept

Encapsulation

Encapsulation basically means wrapping up of data into one single capsule. The basic example of encapsulation is a class which holds all types of data members.

Data Abstraction

Data abstraction basically means hiding the complexity of the data and displaying the relevant ones to the user. In other words data hiding concept is used and it is achieved with the help of various access modifiers such as public private,protected and default.

Inheritance

Inheritance means inheriting the properties of another class or classes.This ensures data reusability. A class acquiring the properties of another class is called as inheritance.In this concept,there are two types of classes;a base class and a derived class. Inheritance can divided into several categories viz: 1:Single level inheritance 2:Multiple inheritance. 3:Hierarchial inheritance. 4:Hybrid inheritance. 5:Multilevel inheritance.

Polymorphism

Polymorphism is a concept which simply states "one name multiple forms". Polymorphism is of two types: 1:Compile time polymorphism(a.k.a early binding and static binding). 2:Runtime polymorphism(a.k.a late binding and dynamic binding). Compile time polymorphism is done by method overloading or operator overloading and runtime polymorphism is done by method overriding and by using virtual function.

Thread is a basic unit of cpu utilization.They are lightweight processes.
A thread is similar to the sequential programs.A single thread also has a beginning, an end, a sequence, and at any given time during the runtime of the thread there is a single point of execution. However, a thread itself is not a program. It cannot run on its own, but runs within a program.

Difference between delete,truncate and drop keyword in sql

Delete

Delete is a DML command
The DELETE command is used to remove rows from a table. A WHERE clause can be used to only remove some rows. If no WHERE condition is specified, all rows will be removed. After performing a DELETE operation you need to COMMIT or ROLLBACK the transaction to make the change permanent or to undo it. Note that this operation will cause all DELETE triggers on the table to fire.

Truncate

Truncate is DDL command.
TRUNCATE removes all rows from a table. The operation cannot be rolled back and no triggers will be fired. As such, TRUNCATE is faster and doesn't use as much undo space as a DELETE

Drop

Drop is a DDL command.
The DROP command removes a table from the database. All the tables' rows, indexes and privileges will also be removed. No DML triggers will be fired. The operation cannot be rolled back.
In other words it removes all the data from the database along with the schema definition.

Normalization and denormalization concept in sql

Database normalization is the process of organizing the fields and tables of a relational database to minimize redundancy and dependency. Normalization usually compresses large tables into smaller tables. Normalization can categorized into 1NF,2NF,3NF,BCNF.
Denormalization is a reversed process of normalization.But the question arises why denormalization?Because if you want to modify the database schema,rather than making changes to all the smaller(normalized) tables we can just modify the denormalized big table and again normalize it.

Difference between abstract classes and interfaces.

Abstract classes are the classes which are declared as abstract.On the other hand interfaces are just used for declaration.Abstract classes can also have non abstract methods whereas interfaces can't.A concrete class extends an abstract class but implements an interface.Interfaces can extend another interface.A class can implement multiple interfaces,thus following multiple inheritance concept.Whereas in abstract class,a concrete class can extend only one abstract class.

What is garbage collection in java?

Garbage collection (GC) is a form of automatic memory management. The garbage collector, or just collector, attempts to reclaim garbage, or memory occupied by objects that are no longer in use by the program. In java objects are stored in heap.Garbage collection or the collector thread frees the objects which are no longer in use from the heap space.It is implicitly called by invoking the finalize() method.The programmer can also explicitly call it by using the system.gc() or runtime.gc() method.

What is the difference between ip and tcp/ip protocol?

The difference is that TCP is responsible for the data delivery of a packet and IP is responsible for the logical addressing. In other words, IP obtains the address and TCP guarantees delivery of data to that address.
